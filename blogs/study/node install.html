# nodejs下载疑难杂症

今天给旧电脑上下载nodejs，可折磨死我了，各种报错

一开始是使用包管理器apt下载，然后发现版本太老了
那我就在官网上下载二进制文件自己安装
安装过程似乎很顺利，但在安装yarn时失败了，一直提示npm错误啥的
那我干脆删了重新下，换一种方式

这时我想起来我服务器上是用nvm来下载nodejs的
那我就去下个nvm吧
安装命令是：

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

好嘛，执行完一点反应也没有，没有报错我还以为成功了
输入nvm一试，没成功
试试另一条命令

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

报错，提示连接失败

那再试试别的下载方法
我选择手动安装，把仓库克隆过来手动配置环境变量
一切似乎很顺利
就在我以为下载成功的时候
输入下载命令：

nvm install v16.17.0

下载nodejs还是失败，提示未找到命令nvm需要curl或wget才能继续
？？？
之后我又七搞八搞，换镜像源啊，重新下wget啊都试过
最后又把nvm删了
我翻出了最开始nvm的安装命令
这条命令的意思是下载脚本文件并执行
可能之前是旧电脑的网络不行？
那我直接复制文件内容再粘贴到服务器试试？
然后我再执行，下载成功，非常顺利，一切正常！
芜湖终于成功了
