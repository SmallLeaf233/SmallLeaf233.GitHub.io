# 对于笔记本副屏方案的尝试

我的旧笔记本一直放在那里吃灰，我就想试试能不能用它来作为我现在这台电脑的副屏。
一开始的想法是直接用 HDMI 线连接，但是笔记本的 HDMI 线貌似只支持输出，不支持输入。
就想能不能从软件方面入手。

# SpaceDesk

我首先选择的是 SpaceDesk (opens new window)，这款软件可以使用网络或 USB 将其他设备作为 Windows 的虚拟显示屏。
使用下来没什么问题，也是我后面尝试下来后认为最简单最好用的。
就是画质和延迟差强人意，或许提高网络速度能解决这个问题吧。

# moonlight

由于无法忍受如此高的延迟，我又尝试了 moonlight。
这是一款开源的游戏串流软件，是 NVIDIA 的 GameStream 协议的开源实现。
它不仅可以串流游戏，还可以通过使用远程桌面连接来串流桌面，实现电脑的副屏。
虽然延迟和画质相比SpaceDesk有很大的提升，但是使用过程中还是有一些问题。
比如直接串流出来的是电脑桌面的复制而不是扩展。
教学视频 (opens new window)中提到可以先设置为仅第二屏幕，然后再改为扩展就能就能进入真正的扩展模式。
但是我设置过后依然还是电脑桌面的复制，而且在电脑的显示设置中也没有显示第二屏幕。
视频中提到了可以试试使用显卡欺骗器来创建一个虚拟的显示屏
我不想花这个钱，所以想着有没有其他的替代方案，比如说由软件实现的虚拟显示器。

废了一番周折，我找到了 Amyuni (opens new window) ，该程序可以注册最多四个虚拟显示器。
在注册了虚拟显示器后，在设置中也能成功看到第二屏幕了。
我试着在设置的投影选项中选择了仅第二屏幕，然后……我的笔记本就黑了，我承认我慌了。
第二屏幕就是软件虚拟的屏幕，它并不存在。
此时我有两个选择：在黑屏的情况下把设置改回去，或者重启试试（重启可解千愁）。
我觉得我并没有那个实力能在黑屏的情况下设置回去，我选择了重启。
所幸重启是有效的，还好之前没有给程序设置开机自启，重启后虚拟显示屏也就没有了。
我再也不瞎搞电脑了。




HDMI显卡欺骗器到了，我觉得我又行了。
插入显卡欺骗器，成功识别到第二屏幕屏，启动 moonlight 进行连接，这里我是在手机上下载 moonlight 进行测试，不过直接串流出来的依然是电脑桌面的复制而不是拓展。
切换为仅第二屏幕，手机上画面冻结，应该是因为之前连接的是主屏幕的信号。
手机重新连接后成功显示第二屏幕的内容，这时电脑也由黑屏转为了空白桌面。
设置中的投影选项自动切换成了拓展，但是所有的窗口只能在手机上显示，也无法将窗口拖动到电脑屏幕。
打开设置看到，较小的屏幕为屏幕1，仅在2上显示。

也就是说电脑似乎将较小的屏幕分配给了原电脑屏幕，将较大的第二屏幕显示在了手机上。
我尝试将第一屏幕设为主显示器，电脑屏幕成功显示，但是画面有问题。
桌面的缩放比例变大了，应用图标也很混乱，在桌面上挤在一起。
我将拓展改为复制，桌面的显示更奇怪了，就像是把屏幕撕裂成了好几块（可惜之前没有截图）
总之这个方案还是存在BUG，我将HDMI显卡诱骗器拔出来，但是桌面依然没有复原，应用图标依然挤在桌面上。

起初我以为是软件将串流出去的画面错误的投放到了屏幕上，导致屏幕分辨率出现了问题。
我尝试重启，但是重启后依然如此，推测可能是程序不小心改了系统里的什么东西。
我看到高级显示器设置中 显示器1：已连接到NVIDIA GeForce RTX 3060 Laptop GPU

我不确定之前这里设置的是什么。
我尝试在设备管理器中禁用显示适配器中的 NVIDIA GeForce RTX 3060 Laptop GPU 。
但是并没有什么效果，只是让电脑变卡了一点，这里貌似是禁用了显卡驱动。
那就应该不是显卡的问题，我在百度中搜索“桌面图标边距变宽”，终于找到了解决方案。
在注册表编辑器中打开这个目录：

计算机\HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics

修改其中的 IconSpacing 和 IconVerticalSpacing 的值为 -1125，这是电脑的默认图标边距。
然后重启电脑，这时桌面图标依然比较大，但是图标边距更为紧凑了。
Ctrl + 鼠标滚轮缩放图标大小，成功还原成了最初的样子。
我真的再也不瞎搞了，副屏什么的还是不弄了，真要弄副屏还是直接买个显示器吧……
